# PyPredictor

Este é um projeto de previsão de preços usando Streamlit e Python.

## Como rodar localmente
1. Clone o repositório
2. Instale as dependências
3. Rode `streamlit run app.py`

## Para acessar o arquivo pela navegador diretamente, basta clicar no link abaixo 
Você pode acessar o app rodando online [clicando aqui] (https://pypredictor-py.streamlit.app/)
